# Задание 04 Простейший калькулятор

В качестве аргументов указываются положительные или отрицательные целые числа - операнды и знак операции.
Вернуть результат вычисления в виде числа с плавающей точкой.

Типы операндов:
```
+  - сложение
-  - вычитание
/  - деление
*  - умножение
```

Пример:
```
a = -25
b = 5
operation = "/"
```

Результат:
-5.0
